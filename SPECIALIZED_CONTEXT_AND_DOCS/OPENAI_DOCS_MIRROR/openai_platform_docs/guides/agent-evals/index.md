Agent evals
===========

Measure agent quality with reproducible evaluations.

Copy page

The OpenAI Platform offers a suite of evaluation tools to help you ensure your agents perform consistently and accurately.

For identifying errors at the workflow-level, we recommend our [trace grading](http://platform.openai.com/docs/guides/trace-grading) functionality.

For an easy way to build and iterate on your evals, we recommend exploring [Datasets](http://platform.openai.com/docs/guides/evaluation-getting-started).

If you need advanced features such as evaluation against external models, want to interact with your eval runs via API, or want to run evaluations on a larger scale, consider using [Evals](http://platform.openai.com/docs/guides/evals) instead.
