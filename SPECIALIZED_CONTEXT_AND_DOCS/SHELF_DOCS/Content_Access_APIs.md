## Page Info

> Get page info with current Gem with its children and application settings.\
> \
> We sort the list of \`folders\` (type \`Directory\`) or \`gems\` (types: \`Note\`, \`Decision Tree\`) by \`title\` and then by \`createdAt\`, both in ascending order.\
> \
> Returns maximum 1000 \`folders\` and 1000 \`gems\`, 3 \`gems\` for each folder first direct then more nested in case current gem type is \`Self-Service Library\` or \`Directory\`.\
> \
> You should pass \`X-Shelf-Slug\` to detect gem ID and language or \`X-Shelf-Gem-ID\` and \`X-Shelf-Gem-Lang\` or \`X-Shelf-Custom-Slug-Gem-ID\` and \`X-Shelf-Custom-Slug-Gem-Lang\`.\
> \
> \`X-Shelf-Custom-Slug-Gem-ID\` can't be equal \`X-Shelf-Library-ID\` because library can't have custom slug.\
>
